# React Single Page Application

This is a sample React application that can be used to demonstrate WebApp deployments of WSO2 Developer Platform.

## Deploy In WSO2 Developer Platform

- Fork this repository.
- Please refer to the documentation under the [Develop a Web Application](https://wso2.com/choreo/docs/develop-components/develop-a-web-application/) section to learn how to deploy the application.

### Use the following configuration when creating this component in WSO2 Developer Platform:

- Build Pack: **React**
- Project Directory: `react-single-page-app`
- Build Command: `npm run build`
- Build output directory: `build`
- Node Version: `18`
